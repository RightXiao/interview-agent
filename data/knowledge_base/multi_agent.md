# Multi-Agent Collaboration

Multi-agent collaboration divides work across role-specific agents. A coordinator agent can route the task, while specialist agents focus on explanation, interview questions, review, or planning.

This project uses a centralized coordinator pattern. It is easier for beginners to understand than fully decentralized agent debate, and it is easier to test.

Interview talking point: explain why centralized coordination is enough for version 1 and how it could be extended later.
