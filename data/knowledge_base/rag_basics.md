# RAG Basics

Retrieval augmented generation means retrieving relevant knowledge before asking the model to answer. It reduces hallucination and allows the system to answer with local or private documents.

This project supports built-in Markdown documents and user imported Markdown, text, and PDF files. PDF import keeps page metadata so answers can cite the original page.

Interview talking point: describe ingestion, chunking, metadata, retrieval, prompt assembly, and source citation.
